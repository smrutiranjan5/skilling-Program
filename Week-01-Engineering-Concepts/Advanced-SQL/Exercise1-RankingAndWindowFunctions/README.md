# Exercise 1 - Ranking and Window Functions

## Objective
Learn and implement SQL Window Functions using:
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- PARTITION BY
- OVER()

## Scenario
An Online Retail Store wants to find the top expensive products in each category using different ranking functions.

## Database
CognizantDB

## Table Used
Products

## SQL Concepts Covered
- Window Functions
- Ranking Functions
- PARTITION BY
- ORDER BY
- Common Table Expression (CTE)

## Output
- Assigned row numbers within each category.
- Compared RANK() and DENSE_RANK().
- Retrieved Top 3 products from every category.
