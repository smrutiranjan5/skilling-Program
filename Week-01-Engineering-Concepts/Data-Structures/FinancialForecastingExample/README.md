# Exercise 7 - Financial Forecasting

## Objective

Predict future financial value using recursion.

## Concept Used

- Recursion
- Financial Forecasting
- Time Complexity Analysis

## Algorithm

1. If years become 0, return current value.
2. Otherwise increase value by growth rate.
3. Call the function recursively.
4. Return predicted value.

## Time Complexity

O(n)

Because one recursive call is made for each year.

## Space Complexity

O(n)

Recursive function uses the call stack.

## Optimization

The recursive solution can be optimized using:

- Iteration
- Dynamic Programming (Memoization)

These approaches avoid excessive recursive calls.
